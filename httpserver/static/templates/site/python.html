Senzing Playground for Python

You can use Jupyter notebooks or the Python SDK to explore Senzing. Senzing SDK methods are documented in the Senzing Software Development Kit (SDK).

Jupyter notebooks

In these demonstrations, Jupyter notebooks have been created to demonstrate Senzing. Use these notebooks as a starting point to explore Senzing. Then, build a notebook tailored to your needs.

If you already have Jupyter installed, start by installing the Python senzing-grpc package into your environment.
Example: 

    python3 -m pip install --upgrade senzing-grpc
Then download and run the following Jupyter notebooks:
  1. senzing_hello_world.ipynb - A simple test of connectivity to Senzing engine.
  2. load_senzing_truthsets.ipynb - Load and query the Senzing truth-set-data.
  3. load_user_data.ipynb - Load custom data.

If you want to see the Jupyter Notebooks in action without installing Jupyter, the Jupyter Lab demonstration consists of tutorials in the following Jupyter Notebooks:

  1. Senzing Hello World - A simple test of connectivity to Senzing engine.
  2. Load Senzing truth-sets - Load and query the Senzing truth-set-data.
  3. Load user data - Load custom data.

Senzing Python SDK demonstration

In these demonstrations, Senzing's Python Software Development Kit for gRPC (SDK) runs natively on a Linux, macOS, or Windows workstation.

Hint: If the senzing-playground docker container has been use in a prior demonstration, restart the docker container for best results.

Prepare your python environment by installing the Python senzing-grpc package into your environment.
Example: 


    python3 -m pip install --upgrade senzing-grpc

Download and run any of the following files:

  1. Senzing Hello World - A simple test of connectivity to Senzing engine.
  2. Load Senzing truth-sets - Load and query the Senzing truth-set-data.
  3. Load user data - Load custom data.
  4. Show method help - Show how to display help.
