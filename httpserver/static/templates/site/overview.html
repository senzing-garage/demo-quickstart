With the Senzing Playground, you are able to explore Senzing from the convenience of your workstation.

Jupyter Lab demonstration

In this demonstration, Jupyter notebooks have been created to demonstrate Senzing.

The Jupyter Lab demonstration consists of tutorials in the following Jupyter Notebooks:

  1. Senzing Hello World - A simple test of connectivity to Senzing engine.
  2. Load Senzing truth-sets - Load and query the Senzing truth-set-data.
  3. Load user data - Load custom data.
  4. Show method help - Show how to display help.

Senzing Python SDK demonstration

In this demonstration, Senzing's Python Software Development Kit for gRPC (SDK) runs natively on a Linux, macOS, or Windows workstation.

The Senzing Python SDK demonstration consists of the following tutorials:

  1. Senzing Hello World - A simple test of connectivity to Senzing engine.
  2. Load Senzing truth-sets - Load and query the Senzing truth-set-data.
  3. Load user data - Load custom data.
  4. Show method help - Show how to display help.
