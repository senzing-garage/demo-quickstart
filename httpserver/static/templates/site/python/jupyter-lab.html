Using my Jupyter Lab

This exercise requires that you already have Jupyter Lab installed on your computer.

Download and run the following Jupyter notebooks in your Jupyter Lab:

  1. senzing_hello_world.ipynb - A simple test of connectivity to Senzing engine.
  2. senzing_load_truthsets.ipynb - Load and query the Senzing truth-set-data.
  3. senzing_load_user_data.ipynb - Load custom data.
