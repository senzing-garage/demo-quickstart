Using personal Jupyter Lab

Add the Python senzing-grpc package into your environment.
Example: 


    python3 -m pip install --upgrade senzing-grpc

Then download and run the following Jupyter notebooks in your Jupyter Lab:

  1. senzing_hello_world.ipynb - A simple test of connectivity to Senzing engine.
  2. senzing_load_truthsets.ipynb - Load and query the Senzing truth-set-data.
  3. senzing_load_user_data.ipynb - Load custom data.
